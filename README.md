# ✈️ ViagensEu - Landing Page de Turismo

Bem-vindo ao repositório da **ViagensEu**, uma landing page moderna, responsiva e profissional desenvolvida para uma agência de turismo fictícia especializada em destinos europeus.

Este projeto foi construído focando em **boas práticas de UI/UX, acessibilidade (a11y), arquitetura CSS avançada e JavaScript moderno focado em performance.**

## ✨ Funcionalidades e Destaques

- **Design Responsivo Avançado:** Utilização de CSS Grid e Flexbox para adaptação fluida em dispositivos móveis, tablets e desktops sem dependência excessiva de Media Queries.
- **Animações Performáticas:** Implementação da API `Intersection Observer` via JavaScript para criar efeitos de *fade-in* (surgimento suave) apenas quando os elementos entram na área visível da tela.
- **ScrollSpy Dinâmico:** O menu de navegação identifica automaticamente a seção que o usuário está visualizando e destaca o link correspondente.
- **Glassmorphism:** Efeito moderno de vidro fosco no cabeçalho fixo de navegação.
- **Acessibilidade Aprimorada:** Uso de tags semânticas HTML5, atributos `alt` em imagens, tags `<figure>` e estados de `:focus-visible` para navegação por teclado.
- **Arquitetura CSS:** Utilização de *Custom Properties* (Variáveis Globais) para fácil tematização e manutenção do código.

## 🛠️ Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando tecnologias web nativas (Vanilla), sem o uso de frameworks externos:

- **HTML5** (Semântica e Acessibilidade)
- **CSS3** (Grid, Flexbox, Variáveis, Animações e `scroll-behavior`)
- **JavaScript (ES6+)** (DOM Manipulation, Event Listeners, Intersection Observer)
